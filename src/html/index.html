But Task, Activity, Skill: What's what?!

Bamboozled doggo
Task is specifically what you did,
Activity is: If you did two similar tasks, how would you group them together?
Skills are what it makes you better at.
Tip: Hover over inputs on the main screen for examples!
Humpty
For example, if you learnt an algorithm while taking a Java x coding course:
Learnt an algorithm is the task, x coding course is the activity, and Java and Algorithms are the skills.
